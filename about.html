About Fira Code

Empowering developers with beautiful, functional typography

About Us

Fira Code is a free monospaced font specifically designed for programmers and developers. Born from the Fira Mono font family, Fira Code extends the original typeface with programming ligatures that enhance code readability and aesthetics.

Our font has been adopted by thousands of developers worldwide, becoming one of the most popular coding fonts in the developer community. It's available across all major platforms and integrates seamlessly with popular code editors and IDEs.

The project is open-source and community-driven, with regular updates and improvements based on user feedback. We believe that beautiful code starts with beautiful typography, and Fira Code delivers exactly that.

Our Mission

To provide developers with a free, high-quality monospaced font that makes code more readable and visually appealing through the use of programming ligatures and thoughtful design.

We aim to:

  • Improve code readability through intelligent ligatures
  • Maintain compatibility across all major platforms and editors
  • Keep the font free and open-source for everyone
  • Continuously improve based on community feedback
  • Support a wide range of programming languages and symbols
  • Make programming more accessible and enjoyable
